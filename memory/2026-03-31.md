### Blocking Issues

#### Angular 17+ Compiler Limitations
1. **Cannot exclude Angular metadata files**
   - All files with `@Component`/`@Directive` decorators must be compiled
   - Cannot selectively compile only working spec files

2. **Template validation cannot be disabled**
   - `NG8001` (unknown element)
   - `NG8002` (unknown property)
   - `NO_ERRORS_SCHEMA`/`CUSTOM_ELEMENTS_SCHEMA` only work at runtime, not during compilation
