Scrollbar control allows scrolling to where the specified element.

## API

| Method            | Property                           | Description       |
| ----------------- | ---------------------------------- | ----------------- |
| `scrollToElement` | `element?: Element, topOffset = 0` | Scroll to element |
| `scrollToTop`     | `topOffset = 0`                    | Scroll to top     |
