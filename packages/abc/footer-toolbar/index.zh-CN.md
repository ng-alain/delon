固定在底部的工具栏。

## 何时使用

固定在内容区域的底部，不随滚动条移动，常用于长页面的数据搜集和提交工作。

## API

### footer-toolbar

| 成员 | 说明 | 类型 | 默认值 |
|----|----|----|-----|
| `ng-content` | 工具栏内容，向右对齐 | `-` | - |
| `[errorCollect]` | 是否需要 `error-collect`，务必包裹在 `<form>` 元素内 | `boolean` | `false` |
| `[extra]` | 额外信息，向左对齐 | `string,TemplateRef<void>` | - |
