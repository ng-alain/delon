一组用户头像，常用在项目/团队成员列表。可通过设置 `size` 属性来指定头像大小。

## API

### avatar-list

| 参数     | 说明     | 类型                       | 默认值    |
| -------- | -------- | -------------------------- | --------- |
| `[size]` | 头像大小 | `large,small,mini,default` | `default` |

### avatar-list-item

| 参数     | 说明         | 类型     | 默认值 |
| -------- | ------------ | -------- | ------ |
| `[tips]` | 头像展示文案 | `string` | -      |
| `[src]`  | 头像图片连接 | `string` | -      |
| `[text]` | 文本类头像   | `string` | -      |
| `[icon]` | 图标类型     | `string` | -      |
