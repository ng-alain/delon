一种进度展示方式，可以更直观的展示当前的进展情况，通常也可表示占比。

## API

| 参数      | 说明                                      | 类型         | 默认值 |
|----------|------------------------------------------|-------------|-------|
| `[title]` | 图表标题 | `string` | - |
| `[height]` | 图表高度 | `number` | - |
| `[color]` | 图表颜色 | `string` | `#2F9CFF` |
| `[bgColor]` | 图表北京颜色 | `string` | `#F0F2F5` |
| `[percent]` | 进度比例 | `number` | - |
| `[format]` | 坐标轴格式 | `Function` | - |
