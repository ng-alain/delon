用于显示跟速度相关图形再适合不过。

## API

### g2-mini-progress

| 参数 | 说明 | 类型 | 默认值 |
|----|----|----|-----|
| `[target]` | 目标比例 | `number` | - |
| `[color]` | 进度条颜色 | `string` | - |
| `[strokeWidth]` | 进度条高度 | `number` | - |
| `[percent]` | 进度比例 | `number` | - |
